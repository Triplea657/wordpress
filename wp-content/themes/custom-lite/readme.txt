## How to install

* Unzip and then copy the layout-builder directory to /wp-content/themes/ in your WordPress installation. 
* Activate from admin menu. 

## How to use

* Theme options placed at "Appearance >> Customize"
* Animation options can be found at  "Customise >> Animation"
* Custom Lite works with WordPress 4.2 and higher. Be sure you use latest version of WordPress.
